

In this lesson we collect the 3D positions of an ArUco marker and save the (x,y) coordinates in a map file.


###

* mapping_with_aruco_markers.py

You can pass as arguments the marker code that will be used for building the map and the map file name.
The intelligent space will use the service of ArUco detector to localize the marker and then save its position in a map file.
  
###

* plot_map.py
  Plot the file given as an argument to the program
